## 三、四个实验

### 实验 1：Sine Wave Studio
**副标题**：Market Cycles Are Predictable. Your Reaction to Them Is Not.

**对应 tag**：`crypto` / `bitcoin` / `trading`

**动画**：
- 全宽 canvas，绘制 1-3 条正弦波叠加
- 波形颜色：品牌紫渐变，移动光点沿波峰流动
- 可选：叠加两条波，展示「合成波」= 市场是多个周期叠加

**数学原理展示**：
```
y = A · sin(ωx + φ)

A = 振幅（市场波动幅度）
ω = 角频率（周期长短）
φ = 相位（入场时机）
```

**交互控制**：
| 参数 | 默认值 | 说明 |
|------|-------|------|
| 振幅 A | 20px | 越大代表波动越剧烈 |
| 频率 ω | 3 | 越大周期越密 |
| 相位 φ | 0 | 滑动感受入场时机变化 |
| 波数 | 1-3 | 叠加波感受合成效果 |

**真实数据钩子（可选接入）**：
- 振幅 = BTC 24h 涨跌幅（来自 market-data.py）
- 颜色亮度 = F&G 指数归一化

**底部引用**：
> "买预期，卖事实。行情是预期的总和，不是对事实做反应。" — Rex Liu  
> [→ 相关文章：SNEK Structure Analysis / How I Got Rekt in Crypto]

### 实验 2：Neural Pulse
**副标题**：Intelligence Is Connection, Not Computation.

**对应 tag**：`ai` / `agent` / `vibe-coding`

**动画**：
- 可配置节点数（5-20）的图，随机布局
- 脉冲从一个节点出发，沿边传播，触达节点时短暂高亮
- 节点大小 = degree（连接数越多越大）

**数学原理展示**：
```
图论基础：G = (V, E)
V = 节点（思维/工具/人）
E = 连接（信息流动路径）

PageRank 启示：
重要性 ≠ 自身能力
重要性 = 被高价值节点连接的次数
```

**交互控制**：
| 参数 | 默认值 | 说明 |
|------|-------|------|
| 节点数 | 11 | 5-25 |
| 脉冲速度 | 1x | 0.5x - 3x |
| 连接密度 | 中 | 稀疏/中/密集 |
| 布局 | 随机 | 随机/圆形/分层 |

**底部引用**：
> "AI 不会取代你，用 AI 的人会取代你。关键是成为高度连接的节点。" — Rex Liu

### 实验 3：Compound Growth
**副标题**：The 8th Wonder of the World, Visualized.

**对应 tag**：`investment` / `wealth` / `finance`

**动画**：
- 折线图从左向右绘制，展示复利增长曲线 vs 线性增长对比
- 两条线：`线性增长（蓝灰）` vs `复利曲线（品牌紫）`
- 差距随时间越来越大，右侧有实时数字跳动

**数学原理展示**：
```
复利公式：FV = PV · (1 + r)ⁿ

PV = 初始金额
r  = 年化收益率
n  = 年数

线性增长：FV = PV · (1 + r·n)
差距 = (1+r)ⁿ - (1+r·n)
      ↑ 时间越长，差距越大
```

**交互控制**（预填 Rex 的真实目标）：
| 参数 | 默认值 | 范围 |
|------|-------|------|
| 初始金额 PV | ¥100万 | 1万-1000万 |
| 年化收益率 r | 8% | 1%-30% |
| 时间 n | 20年 | 1-40年 |

**底部实时显示**：
```
20 年后
线性增长：¥260万
复利增长：¥466万
差距：¥206万（+79%）
```

**底部引用**：
> "Stay Invested in the Game. 时间是最好的杠杆。" — Rex Liu  
> [→ 相关文章：Gold ETF Trade Review / Investment Philosophy]

### 实验 4：Bezier Journey
**副标题**：Life Is Not a Straight Line. It's Pulled by Invisible Control Points.

**对应 tag**：`travel` / `lifestyle` / `life`

**动画**：
- 三阶贝塞尔曲线，控制点可拖动
- 一个点沿曲线匀速移动，留下轨迹尾迹
- 右侧罗盘随曲线切线方向转动（展示方向感）

**数学原理展示**：
```
三阶贝塞尔曲线：
B(t) = (1-t)³P₀ + 3(1-t)²tP₁ + 3(1-t)t²P₂ + t³P₃
t ∈ [0, 1]

P₀ = 起点（当下处境）
P₁,P₂ = 控制点（机遇/选择/牵引力）
P₃ = 终点（目标）

真相：改变控制点，终点不变，但路径完全不同。
```

**交互控制**：
- **可拖动控制点** P₁、P₂
- 移动时曲线实时更新
- 拖动感受「同样的起点和终点，不同的选择导致不同的人生轨迹」

**底部引用**：
> "我们选择的不是终点，是控制点。" — Rex Liu  
> [→ 相关文章：Dreams from My Parents / Anniversary to Shanghai]
