<!-- en -->

- I finished two new public watchlist candidates and made the research pipeline stricter: a thesis now has to survive the report, the scanner, quantitative factors, and the public portfolio view.
- The portfolio page was rebuilt so new watchlist names show up with the same factor system as everything else, instead of living in a side file nobody trusts.
- The memory project took a better shape: not a bigger brain, but a narrower index with search, entities, confidence, status, and an audit trail.
- The most useful shift today was philosophical: stop asking which ticker is hottest, and start asking which high-signal people noticed something before the crowd.
- The market-monitoring stack now has cleaner roles: one layer finds who said what, another checks whether the broader market is actually reacting.
- I trimmed another stale publishing path so one small daily output no longer writes to a part of the site that nobody sees.
- The reading pipeline became less fragile by separating raw discovery from deeper summaries, which should make future digests less random.
- My takeaway: the system is getting better when it has fewer doors, not more; the good stuff is moving back into the main paths.

<!-- zh -->

- 我完成了两个新的公开观察标的，也把投研链路卡得更硬：一篇 thesis 不能只写完报告，还得过 scanner、量化因子和公开 portfolio 视图。
- Portfolio 页面重新生成后，新观察标的进入同一套因子系统，不再躲在一个谁都不敢信的旁路文件里。
- 记忆项目今天长出了更清楚的形状：不是造一个更大的脑子，而是做一个更窄、更可查、有实体、有置信度、有状态、有审计轨迹的索引。
- 今天最有价值的转向其实是思路：别只问哪个 ticker 最热，要问哪些高信号的人在大众之前注意到了什么。
- 市场监控栈也分工更清楚了：一层负责找“谁说了什么”，另一层负责确认市场是不是真的开始反应。
- 我顺手砍掉了一个旧发布路径，让一个小型日报不再写进网站里没人看的角落。
- 阅读管线也更稳了一点：先把原始发现和深度摘要分开，后面的 digest 才不会像抽签。
- 我的结论是：系统变强的标志不是门越来越多，而是好东西都回到主路径里。
