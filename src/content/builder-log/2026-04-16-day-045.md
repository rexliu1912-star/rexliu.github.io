- Shipped the public `/portfolio` page with holdings, funnel stages, trade replay, roundtable notes, and a macro thermometer, which finally made the research stack visible instead of buried in docs.

- Tightened the data side behind that page so it is not just a pretty shell, and the next real job is turning the daily snapshot into a stable cron on the Mac Mini.

- Picked `unsloth/Qwen3.5-35B-A3B Q4_K_M` as the local heavy tier after a lot of noise, mostly because the best model is not the one with the best benchmark, it is the one that fits the workflow without adding drag.

- Watched the script-based download path fall apart under proxy resets, which was a useful reminder that sometimes the boring native UI is the smarter tool.

- Saw both Alpha Radar and daily reading point to the same idea: AI competition is sliding away from pure model rankings and toward private infrastructure, workflow control, and data boundaries.

- Kept the daily intel chain running with the SNEK briefing, digest sync, and workload refresh, so the system still produced while bigger pieces were moving.

- Noticed one annoying truth again: a task marked done is not the same thing as a task fully delivered, especially when the last notification hop gets blocked.

- Had one of those builder moments I actually like, where the work stops feeling like feature collecting and starts feeling like the stack beneath the interface is finally becoming real.

<!-- zh -->
- 把公开的 `/portfolio` 页面正式挂上去了，持仓、阶段漏斗、交易回放、圆桌结论和宏观温度计都放进来了，研究系统第一次不像藏在文档里。

- 也顺手把页面后面的数据链路收紧了一层，不让它只是个好看的壳子。下一步最实在的事，就是把 Mac Mini 上的 daily snapshot 补成稳定 cron。

- 本地 heavy tier 最后拍板成 `unsloth/Qwen3.5-35B-A3B Q4_K_M`，这次我更确认了一件事，最值得选的模型不是跑分最好那个，而是最不打断工作流那个。

- 代理下载大模型这条路今天也算彻底试明白了，脚本一遍遍断连。某些时候，老老实实用原生 GUI，反而比花里胡哨的自动化更聪明。

- Alpha Radar 和 daily reading 今天指向了同一条线：AI 的竞争焦点正在从模型榜单慢慢滑向私有基础设施、工作流控制权和数据边界。

- 日常情报链路也没停，SNEK 简报、digest 同步、workload 刷新都照常跑着，说明系统在搬大石头的时候，底盘还算稳。

- 另外又被提醒了一次，任务显示 done，不等于真的交付完。特别是最后一跳通知被拦的时候，假完成感最容易骗人。

- 今天有一个我挺喜欢的 builder 时刻，事情终于不只是堆功能了，而是那层藏在界面下面的底座，开始一点点长成真的系统。
