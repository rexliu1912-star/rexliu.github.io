- Rebuilt the agent rules system — merged three overlapping rule families (review, before-you-act, retro/investigate) into single files. Went from 20 files to 13, word count dropped 44%. Models no longer need to guess which rule to read.

- Connected a 5-year personal note archive to the content pipeline. Everything that was private stays private — drafts go through review before publishing. But now there's actual material to work with, not just templates.

- Found 11 cron jobs with missing delivery configs. Fixed them in one pass. Also patched a script bug that had been silently dropping recurring signals from the weekly research sweep.

- Lucky hit Day 21 — a 3-week milestone. Used an AI-generated storybook on the train to review three weeks of English vocabulary. Found a pattern that works: watch an episode, then do a related craft the next day. Input → output, same topic, memory sticks.

- Relationship Week 3 closes with the highest weekly average score yet. The one thing that kept dropping the score (losing patience with the kid) is now the single focus for next week.
