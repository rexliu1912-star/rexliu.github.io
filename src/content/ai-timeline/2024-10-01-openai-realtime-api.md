OpenAI launched the Realtime API in public beta on October 1, 2024, enabling developers to build low-latency, speech-to-speech conversational experiences powered by GPT-4o. The API supports six preset voices and handles interruptions natively, enabling more natural human-AI dialogue. It unlocks a new class of voice agent applications without requiring developers to stitch together separate ASR, LLM, and TTS pipelines. The Realtime API marked OpenAI's first developer-facing product for building persistent, interactive voice agents at scale.
