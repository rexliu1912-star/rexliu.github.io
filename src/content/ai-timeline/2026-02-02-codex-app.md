OpenAI launched the Codex desktop app (macOS) on February 2, 2026, a focused development environment for running and managing Codex agent threads in parallel with built-in worktree support, automations, and Git integration. Built with Electron for cross-platform expansion, the app functions as a command center for developers working with Codex agents across multiple tasks simultaneously. It arrived alongside GPT-5.2-Codex and GPT-5.3-Codex, positioning the Codex ecosystem as a comprehensive AI-native IDE alternative. Windows support was announced as "coming very soon" at launch.
