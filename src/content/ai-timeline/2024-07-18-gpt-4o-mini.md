OpenAI released GPT-4o mini, a highly cost-efficient small model designed to replace GPT-3.5 Turbo across most use cases. At $0.15 per million input tokens, it was far cheaper than previous models while outperforming GPT-4 Turbo on key benchmarks. GPT-4o mini supported both text and vision inputs and featured a 128K context window. The release democratized access to powerful AI capabilities and became the default model for high-volume API applications.
