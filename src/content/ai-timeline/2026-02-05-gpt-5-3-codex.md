OpenAI released GPT-5.3-Codex on February 5, 2026, advancing both frontier coding performance and reasoning/professional knowledge in a single model that is also 25% faster than GPT-5.2-Codex. The model set new benchmarks for agentic software engineering, enabling Codex to tackle longer, more complex real-world engineering workflows with improved judgment on ambiguous tasks. It launched three days after the Codex desktop app, cementing OpenAI's position in the rapidly growing AI coding agent market. GPT-5.3-Codex marked OpenAI's most direct challenge to Anthropic Claude Code and Google Jules.
