Anthropic launches Claude Code Security, a new capability built into Claude Code on the web, in a limited research preview for Enterprise and Team customers (with expedited access for open-source maintainers). Unlike rule-based static analysis tools, it reasons about code like a human security researcher — tracing data flow, understanding component interactions, and catching complex vulnerabilities such as broken access control and business logic flaws. Findings go through a multi-stage verification process to filter false positives before reaching analysts.

**So what?** AI-generated code is flooding codebases faster than human reviewers can audit it. Anthropic is positioning Claude Code Security as the defense layer against AI-enabled attacks — using the same frontier reasoning capabilities that could help attackers to instead put power in defenders' hands. Releasing as a research preview first is deliberate: the dual-use risk is real, and they want to refine deployment guardrails before scaling.
