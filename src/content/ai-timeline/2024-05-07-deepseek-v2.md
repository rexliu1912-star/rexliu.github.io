DeepSeek released DeepSeek-V2 on May 7, 2024, a powerful 236B parameter Mixture-of-Experts (MoE) model that activated only 21B parameters per token. The architecture delivered GPT-4-level performance at a fraction of the inference cost, with API pricing 75% cheaper than GPT-4 Turbo. DeepSeek-V2 achieved state-of-the-art performance among open-source models on reasoning and coding benchmarks. The release demonstrated that Chinese AI labs could build frontier-level models using innovative architectures, foreshadowing DeepSeek's later global impact.
